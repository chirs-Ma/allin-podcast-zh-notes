# All-In Podcast 中文可视化笔记

将 [All-In Podcast](https://www.youtube.com/@allin) 的精彩内容以中文呈现，快速获取科技、商业与投资的前沿洞察。

## 在线访问

GitHub Pages: https://chirs-ma.github.io/allin-podcast-zh-notes/

## 项目说明

本项目是一个纯静态网站，托管在 GitHub Pages 上：

- **纯 HTML + CSS**，无需构建工具，零依赖
- 所有样式内联在 HTML 中，单文件即可运行
- 响应式设计，适配桌面端和移动端
- 支持实时搜索过滤

## 内容来源

内容基于 All-In Podcast（由 Jason Calacanis、Chamath Palihapitiya、David Sacks、David Friedberg 主持）的每期讨论，整理为中文可视化笔记，包含：

- 核心要点速览
- 按时间线分段整理
- 四位嘉宾的观点对照
- 双语引语
- 背景术语解释

## 本地使用

```bash
# 克隆仓库
git clone https://github.com/chirs-Ma/allin-podcast-zh-notes.git
cd allin-podcast-zh-notes

# 本地预览（任意静态服务器）
python3 -m http.server 8080
# 然后打开 http://localhost:8080
```
